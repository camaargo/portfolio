Meus Projetos

  • Do Campo à Mesa

    O projeto Do Campo à Mesa foi desenvolvido para o Agrinho 2025 com foco em conscientizar sobre práticas agrícolas e sustentabilidade por meio de perguntas interativas. O maior desafio foi trabalhar a interação visual para deixar a experiência clara e atrativa. Com ele, aprendi a unir conteúdo educativo a uma interface intuitiva, além de aprimorar minhas habilidades em HTML, CSS e JavaScript.

  • Lista de Tarefas

    O projeto Lista de Tarefas foi desenvolvido com o objetivo de ajudar na organização das atividades do dia a dia de forma prática e simples. Ele permite adicionar novas tarefas, marcar como concluídas e removê-las quando necessário, proporcionando uma experiência intuitiva para o usuário. Para sua construção, utilizei HTML para a estrutura da página, CSS para a estilização e design, e JavaScript para a implementação da lógica principal, responsável por toda a interação dinâmica da aplicação.
